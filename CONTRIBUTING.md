# Contributing

Thanks for your interest in contributing.

## Development

```bash
pnpm install
pnpm build
```

## Pull Requests

- Keep changes focused and minimal.
- Update tests/docs/changelog when behavior changes.
- Ensure `pnpm build` passes before opening PR.

## Commit Messages

Recommended style:

- `feat: ...`
- `fix: ...`
- `docs: ...`
- `chore: ...`
