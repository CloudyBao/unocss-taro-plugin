# Code of Conduct

## Our Pledge

We are committed to providing a friendly, safe, and welcoming environment for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus on technical discussion and collaboration.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Publishing private information without consent.
- Any abusive or threatening language.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

For serious issues, contact maintainers via repository security channel.
